# Design Skills CLI

AI-powered design thinking toolkit for designers & teams.

## 🚀 Install

curl -fsSL https://raw.githubusercontent.com/muthuveerapandi-s-2874/Design-skills/main/install.sh | bash

## ⚡ Usage

mytool ui-thinking
mytool design-lang
mytool theme

## 💡 What this does

- Standardizes design thinking
- Converts ideas → structured frameworks
- Helps designers work faster with AI
